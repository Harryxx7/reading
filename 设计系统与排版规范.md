# 66句长难句笔记 · 暖色纸质风设计系统与交互规范 (Design System & UX Spec)

本指南详细总结了本项目的 **“暖色纸质风”视觉美学** 与 **“一句一页”独立滚动交互架构**。您可以直接将此设计系统与排版规范复用到您的其他网页、笔记或应用程序项目中。

---

## 🎨 一、 色彩系统 (Color Palette Tokens)

本系统采用极富质感的**暖偏色调**代替生硬的黑白，大幅降低长时间阅读的视觉疲劳感。主要通过 CSS 变量在 `:root` 和 `[data-theme="dark"]` 中定义。

### 1. 浅色模式 (Light Mode - 暖色宣纸风)
* **核心美学**：模拟高质量宣纸/高档书籍内页的触感，给人温暖、静谧、沉浸的教研阅读氛围。

```css
:root {
  /* 背景与卡片 */
  --bg: #FBF9F4;             /* 暖米白底色（如温润宣纸） */
  --bg-elevated: #FFFFFF;    /* 浮动层/输入框纯白 */
  --card: #F2EBDC;           /* 卡片浅赭石色 */
  --card-strong: #ECE2CC;    /* hover/高亮层卡片深赭石色 */
  --border: #E8E0D0;         /* 极柔和的细边框线 */
  --border-strong: #D4C9B0;  /* 强边框线 */
  --table-alt: #F5F0E5;      /* 表格隔行换色背景 */
  
  /* 文字色彩 */
  --text-primary: #2D2A26;   /* 炭黑/深赫石（比纯黑 #000 柔和得多） */
  --text-secondary: #6B6660; /* 次级辅助文字 */
  --text-muted: #9A9489;     /* 哑光/占位文字 */
  
  /* 语法成分与高亮 */
  --accent-primary: #B85C38;   /* 主题红/赭石红（常用于主句、主谓） */
  --accent-secondary: #2B5A87; /* 沉静蓝（常用于主语、从句） */
  --accent-tertiary: #5B7A4B;  /* 苔藓绿（常用于状语、译文） */
  --accent-purple: #7A5C8F;    /* 哑光紫（常用于修饰成分、定语） */
  --highlight: #FFF1B8;        /* 荧光黄高亮笔 */
  
  /* 深度投影 */
  --shadow: 0 10px 30px rgba(45, 42, 38, 0.04), 0 1px 3px rgba(45, 42, 38, 0.02);
}
```

### 2. 深色护眼模式 (Dark Mode - 温暖黑曜石)
* **核心美学**：拒绝生硬的纯黑 `#000`，采用**带红棕色暖意暗调的黑曜石色**，并在元素周围投射温和的霓虹呼吸微光。

```css
[data-theme="dark"] {
  /* 背景与卡片 */
  --bg: #1F1B17;             /* 暖曜石黑背景 */
  --bg-elevated: #2A241D;    /* 浮动层暗赫石 */
  --card: #2A241D;           /* 卡片深红褐 */
  --card-strong: #332C24;    /* 卡片高亮层 */
  --border: #3A332C;         /* 极暗边框线 */
  --border-strong: #4D4439;  /* 强边框线 */
  --table-alt: #262119;      /* 表格交替行 */
  
  /* 文字色彩 */
  --text-primary: #E8E2D5;   /* 羊皮纸白（降低白字刺眼度） */
  --text-secondary: #B5AEA0; /* 辅助灰 */
  --text-muted: #8A8478;     /* 哑光灰 */
  
  /* 语法高亮（高亮色彩整体做亮色和降低饱和度处理） */
  --accent-primary: #E89568;   /* 柔珊瑚红 */
  --accent-secondary: #7BA8D9; /* 天空蓝 */
  --accent-tertiary: #9CB888;  /* 浅草绿 */
  --accent-purple: #B89BC9;    /* 丁香紫 */
  --highlight: #5C4F1E;        /* 暗金色荧光笔 */
  
  /* 霓虹发光投影 */
  --shadow: 0 10px 40px rgba(0, 0, 0, 0.3), 0 1px 8px rgba(0, 0, 0, 0.2);
}
```

## ✍️ 二、 字体学规范 (Typography System)

在同一段落内对中文和英文使用不同的高保真字体系列：

1. **英文原文（学术沉浸感）**：
   * 必须采用**高档书本印制级的衬线体（Serif）**，让长难句显得极其优雅、舒展，仿佛在阅读原版纸质名著：
   * `font-family: Charter, "Iowan Old Style", "Source Serif Pro", Georgia, serif;`
2. **中文释义与系统 UI（清晰干练）**：
   * 必须采用**现代且极其清晰的无衬线体（Sans-serif）**，防止细线体在中文字号小的时候产生模糊：
   * `font-family: -apple-system, BlinkMacSystemFont, "PingFang SC", "Hiragino Sans GB", "Microsoft YaHei", sans-serif;`

## 🏛 三、 空间布局架构 (Layout Architecture)

这是解决“纵向来回滚动疲劳”的核心交互结构，称为 **“单页视口隔离架构”**：

```
+---------------------------------------------------------+
|                    TOPBAR (56px)                        |
+---------------------------------------------------------+
|  S  |                                                   |
|  I  |  [Card Header]  Word Badges & Mastery Lights      |
|  D  |  +---------------------------------------------+  |
|  E  |  |  Zone Original (Original English Sentence)  |  |
|  B  |  +---------------------------------------------+  |
|  A  |  |  Zone Middle (Flex Dual Columns)            |  |
|  R  |  |  - Left Column: Structure Decode Table      |  |
|     |  |  - Right Column: Vocab & Insights Table     |  |
|  2  |  |  * (Left/Right columns scroll independently)*|  |
|  4  |  +---------------------------------------------+  |
|  0  |  |  Zone Translation (Reference Translation)   |  |
|  p  |  +---------------------------------------------+  |
|  x  |  [Card Footer]  Navigation & Back/Next Arrows    |  |
+-----+---------------------------------------------------+
```

### 1. 桌面端双侧独立滚动限制 (CSS Spec)
为了实现“整页不滚动，只滚动表格”的 App 级体验：
```css
@media (min-width: 1100px) {
  body {
    height: 100vh;
    overflow: hidden; /* 强行切断浏览器外层滚动条 */
  }
  .content {
    height: calc(100vh - var(--topbar-height));
    overflow: hidden;
  }
  .sentence-card {
    height: calc(100vh - var(--topbar-height) - 48px); /* 卡片填满视口 */
    display: flex;
    flex-direction: column;
  }
  .zone-middle {
    flex: 1;
    min-height: 0; /* 极其重要！必须设置 min-height，子列独立滚动条才会生效 */
    display: grid;
    grid-template-columns: 58fr 42fr;
    gap: 28px;
  }
  .zone-middle > .zone {
    height: 100%;
    overflow-y: auto; /* 仅在子列内部显示滚动条 */
  }
}
```

## 💫 四、 高级微交互规范 (Wow Factor Specs)

通过流畅的微交互，彻底让枯燥的静态页面变得“生动且性感”：

### 1. iOS 级弹性阻尼卡片转场 (Spring Page Transition)
当进行翻页或 hash 切换时，卡片不会瞬间闪现，而是采用类似 Keynote 幻灯片的缓入回弹微动画：
```css
@keyframes cardEntrance {
  0% {
    opacity: 0;
    transform: translateY(20px) scale(0.985);
  }
  100% {
    opacity: 1;
    transform: translateY(0) scale(1);
  }
}
.sentence-card.active {
  display: flex;
  animation: cardEntrance 0.45s cubic-bezier(0.34, 1.56, 0.64, 1) forwards; /* 完美的物理阻尼回弹曲线 */
}
```

### 2. 毛玻璃拟物导航与超细边框 (Glassmorphism & Depth)
* **顶栏与侧边栏背景**：采用透明度混合的半透明底色，配合 `20px` 高级毛玻璃滤镜，形成层级通透感：
  `background: color-mix(in srgb, var(--bg) 75%, transparent); backdrop-filter: blur(20px);`
* **超细边框**：全局卡片采用 `0.5px` 极细边框（在高清屏上显示效果极具线条张力，比 `1px` 显得更加 Premium）：
  `border: 0.5px solid var(--border);`

### 3. 自测模式下的“悬停刮刮卡” (Hover-to-Reveal Active Recall)
* 开启“自测模式”时，解析区进入深度模糊，但**鼠标悬停时会柔和解冻恢复清晰**：
```css
/* 自测模糊遮罩 */
.test-masked .zone-middle,
.test-masked .zone-translation {
  filter: blur(14px);
  opacity: 0.15;
  cursor: help;
  transition: filter 0.35s cubic-bezier(0.4, 0, 0.2, 1), opacity 0.35s cubic-bezier(0.4, 0, 0.2, 1);
}
/* 鼠标悬停瞬时透视 */
.test-masked .zone-middle:hover,
.test-masked .zone-translation:hover {
  filter: blur(0);
  opacity: 1;
  cursor: default;
}
```

## 📱 五、 响应式与设备适配规范 (Responsive Device Adaptations)

* **Mac/大屏 (横屏 $\ge 1100px$)**：高度锁定、多栏切分独立滚动、呼吸发光阴影。
* **iPad/手机 (竖屏 $< 1100px$)**：
  * 页面高度恢复自适应滚动 (`height: auto !important; overflow: auto !important;`)。
  * **原句吸顶粘性悬浮**：英文原句区块锁定在顶栏正下方（`position: sticky; top: var(--topbar-height);`），这样即使滑动看最下方的翻译，原句也能随时在屏幕顶部可见。
  * **窄屏防换行重叠**：小屏幕下卡片头部从 `row` 改为 `column`，按钮和标签自适应收缩。
